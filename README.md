## How to use

1. Weigh yourself every morning (before breakfast, after bathroom, barefoot)
2. Log daily data in **logs/** (weight, food, exercises)
3. Update weekly menu in **WEEKLY-MENU.md**
4. Update body measurements monthly in **PROGRESS.md**
5. Track your smart scale data via OKOK International app

## Tools

- Smart scale with OKOK International app (weight, body fat %, visceral fat, muscle %, etc.)
